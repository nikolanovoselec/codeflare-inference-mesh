The override is stored on the node and applied to the desired profiles the node receives in its heartbeat, so the agent renders `--max-vram` at the node's own ceiling instead of the model's global budget. ([REQ-ADM-023](setup-admin.md#req-adm-023-per-node-vram-override))

Create reveals the secret once (click-to-copy); rotate (`POST /api/v1/keys/{id}/rotate`) issues a fresh secret and retires the old one so the previous secret stops authenticating; revoke disables a key immediately. The console mint runs under the admin's Access session — Cloudflare Access injects the admin identity even on the `/api/v1` machine-bypass path, so no stored bearer is needed. ([REQ-ADM-022](setup-admin.md#req-adm-022-api-key-management-console), [REQ-API-001](control-plane-api.md#req-api-001-automation-credentials))
